# A101 SOS - APK Oluşturma Rehberi (GÜNCEL)

Bu uygulamanın A101 SOS modüllerini içeren versiyonunu APK yapmak için aşağıdaki adımları izleyin.

## 1. Hazırlık (Windows Üzerinde)
Öncelikle `main.py` dosyasındaki sunucu adresini kontrol edin. Eğer yerel ağda test edecekseniz `localhost` yerine bilgisayarınızın IP adresini (örn: `192.168.1.XX`) yazmanız gerekebilir.

Ancak sunucu aynı cihazda çalışacaksa `localhost` kalabilir fakat Android cihazlar PC'deki localhost'a erişemez. Bu yüzden **PC IP adresinizi** kullanmanız önerilir.

1.  `baslat.bat` ile uygulamanın bilgisayarda çalıştığından emin olun.
2.  `buildozer.spec` dosyasını güncelledim, gerekli tüm kütüphaneler eklendi.

## 2. Google Colab ile APK Yapma (ÖNERİLEN)
Bilgisayarınızı yormadan Google sunucularında hızlıca APK üretebilirsiniz.

1.  **Google Colab'ı Açın:** [https://colab.research.google.com/](https://colab.research.google.com/)
2.  **Dosyaları Yükleyin:** Sol menüdeki klasör simgesine tıklayıp şu dosyaları sürükleyip bırakın:
    *   `main.py`
    *   `firebase_manager.py` (Eğer kullanılıyorsa)
    *   `buildozer.spec`
    *   `market_bank_db.json` (Veritabanı yapısı için gerekebilir, içi boş olsa da olur)
    *   `requirements.txt`
3.  **Hücreleri Çalıştırın:** Yeni bir kod hücresi açıp şu komutları sırasıyla yapıştırıp çalıştırın:

```bash
# 1. Gerekli Araçları Kur
!pip install buildozer cython==0.29.33
!sudo apt-get update
!sudo apt-get install -y python3-pip build-essential git python3 python3-dev ffmpeg libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev libportmidi-dev libswscale-dev libavformat-dev libavcodec-dev zlib1g-dev

# 2. Buildozer'ı Başlat (Dosyaları yüklediyseniz init'e gerek yok, spec dosyanız var)
# !buildozer init  <-- Bunu yapmayın, kendi spec dosyanızı kullanın.

# 3. APK'yı Derle (Debug Modu)
!buildozer -v android debug
```

4.  **İndirin:** İşlem bitince (15-20dk) sol menüde `bin` klasörü oluşacak. İçindeki `.apk` dosyasını telefonunuza indirip kurun.

## 3. WSL (Windows Subsystem for Linux) İle Yapma
Eğer bilgisayarınızda Ubuntu/WSL kurulu ise:

1.  WSL terminalini açın.
2.  Proje klasörüne gidin: `cd /mnt/c/Users/yunus1/Desktop/satış\ b\ pilan`
3.  Gerekli paketleri kurun (üstteki Colab komutları ile aynı).
4.  `buildozer -v android debug` komutunu çalıştırın.
